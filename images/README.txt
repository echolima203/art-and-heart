PHOTOS FOR YOUR GUEST WEBSITE
=============================

The website automatically shows any photo placed in this "images" folder,
as long as the file name matches exactly (all lowercase, .jpg).

Drop your photos here using these exact names:

  beziers.jpg          -> Hero background (town & cathedral)
  cathedral-sunset.jpg -> Cathedral banner / footer background
  living-room.jpg      -> Featured detail photo
  living-wide.jpg      -> Living & dining room
  salon.jpg            -> Lounge & sofa
  living-cosy.jpg      -> Relax corner
  bedroom1.jpg         -> Bedroom 1 (tropical mural)
  bedroom1-detail.jpg  -> Bedroom details
  bedroom2.jpg         -> Second bedroom
  workspace.jpg        -> Work desk
  courtyard.jpg        -> Private stone courtyard
  courtyard2.jpg       -> Historic stairway
  kitchen.jpg          -> Kitchen
  bathroom.jpg         -> Bathroom
  bathroom2.jpg        -> Bathroom (second view)
  mairie.jpg           -> Place de la Mairie (guide page)
  street.jpg           -> Pedestrian old town (guide page)
  feria.jpg            -> Feria arena (guide page)
  cours-guibal.jpg     -> Cafe terraces
  beach.jpg            -> Valras-Plage beach

TIPS
- Use .jpg files. If yours are .png or .avif, rename/convert them to .jpg
  (most phones and free tools can export as JPG).
- Landscape photos (wider than tall) look best in the gallery.
- Keep each file under ~1 MB so the page loads fast on phones.
- You can add more tiles later by editing index.html (copy one <div class="tile">
  block and point it at a new file name).

Any slot without a matching photo simply shows a soft tile with a label,
so the page always looks tidy even before all photos are added.
